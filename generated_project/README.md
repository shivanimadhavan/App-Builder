# Simple Calculator

A lightweight web-based calculator that performs basic arithmetic operations. It provides an intuitive user interface built with HTML, CSS, and JavaScript, and works across desktop and mobile browsers without any build steps.

---

## Tech Stack
- **HTML** – Structure of the calculator UI.
- **CSS** – Styling and responsive layout.
- **JavaScript** – Core logic for handling button clicks and performing calculations.

---

## Features
- **Basic Operations** – Addition, subtraction, multiplication, and division.
- **Clear & Delete** – Clear the entire expression or delete the last entry.
- **Decimal Support** – Perform calculations with floating‑point numbers.
- **Keyboard Input** – Use your keyboard for numbers and operators (optional enhancement).
- **Responsive Design** – Layout adapts gracefully to different screen sizes, from large desktops to small mobile devices.

---

## Setup Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/simple-calculator.git
   cd simple-calculator
   ```
2. **Open the application**
   - Locate the `index.html` file in the project root.
   - Double‑click it or open it directly in your preferred web browser (Chrome, Firefox, Safari, Edge, etc.).
3. **No build step required** – The project runs purely in the browser; there are no dependencies or compilation steps.

---

## Usage Guide
The calculator UI consists of a display panel and a grid of buttons.

- **Display Panel** – Shows the current expression and the result.
- **Number Buttons (0‑9)** – Enter digits.
- **Operator Buttons (+, –, ×, ÷)** – Choose the arithmetic operation.
- **Decimal (.)** – Add a decimal point for fractional numbers.
- **Equals (=)** – Compute the result of the entered expression.
- **Clear (C)** – Reset the entire expression.
- **Delete (←)** – Remove the last character entered.

### Example Workflow
1. Press `7`, `+`, `3`, then `=` → The display shows `10`.
2. Press `C` to start a new calculation.

## Responsive Design
The calculator uses flexible CSS grid and media queries to adjust its layout:
- On **large screens**, buttons are spaced out with larger touch targets.
- On **mobile devices**, the grid shrinks, button fonts scale down, and the layout stacks to fit the viewport while maintaining usability.
