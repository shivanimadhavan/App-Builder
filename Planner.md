User Prompt: Create a simple calculator web application

# Planner

**Description:**  
A lightweight web application that provides a basic calculator for performing addition, subtraction, multiplication, and division with a clean, responsive user interface.

**Tech Stack:**  
HTML, CSS, JavaScript

**Features**

| # | Feature Description |
|---|---------------------|
| 1 | User-friendly interface with buttons for digits and operations |
| 2 | Display area showing current input and results |
| 3 | Support for addition, subtraction, multiplication, and division |
| 4 | Clear (C) and backspace functionality |
| 5 | Keyboard input support for numbers and operators |
| 6 | Responsive design for desktop and mobile devices |

**Files**

| Path | Purpose |
|------|---------|
| `index.html` | HTML structure of the calculator UI |
| `style.css` | Styling for layout, responsiveness and visual design |
| `app.js` | JavaScript logic for handling input, calculations, and UI updates |
| `README.md` | Project overview, setup instructions, and usage guide |
